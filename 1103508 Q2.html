Q2: How to create a registration form using HTML?

Ans:

1.Initialize the Form : The "form" element can be used to initialize a form that could be a contact, register, or login form on any HTML page.

2. Form Structure : Once we have the "form" element in place, I add many DIVs inside .

3. Add codes inside : Change the color、position ,etc.

4. Form Body : firstname、lastname、email、password 、classes、gender ,etc.
